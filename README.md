# ec2mock
A mock ec2 client written in go.
Currently supported methods:
- DescribeInstances
- DescribeImages
- RunInstances
- CreateTags
- StartInstances
- StopInstances
- TerminateInstances
- WaitUntilInstanceRunningWithContext
- WaitUntilInstanceStoppedWithContext
- WaitUntilInstanceExistsWithContext
